SYSTEM QUALIFICATIONS

Platform Availability & How You Can Play

PC (Windows 10/11)

Release: Launched on October 25, 2024, available through Steam, Battle.net, Microsoft Store, and Game Pass PC.
Internet: A continuous broadband internet connection is required for all game modes due to texture-streaming. Wikipedia


PlayStation (PS5 & PS4)

Platforms: Available on both PS5 and PS4.

PlayStation PS5 Enhancements: Supports 4K visuals, 3D Audio, haptic feedback, and adaptive triggers via DualSense. Call of Duty PS Plus subscription is required for online multiplayer. PlayStation Note: The PS5 version lacks a Platinum trophy, though it includes all 44 trophies—this has disappointed some players.


Xbox (Series X|S & One)

Release Platforms: Available on Xbox Series X|S and Xbox One, including availability through Game Pass on launch. Mobile Devices (Android & iOS) via Cloud Streaming


How to Play: With an Xbox Game Pass Ultimate subscription, you can stream the game via Xbox Cloud Gaming on mobile devices using the Game Pass app (Android) or web browser (iOS). A controller is required.


Supported Devices: Includes mobile phones, PCs, Amazon Fire TV, Meta Quest headsets, and select Samsung TVs.

SYSTEM REQUIREMENTS FOR ALL PLATFORMS

1. PC (Windows)

Minimum

OS: Windows 10 64-bit
• CPU: AMD Ryzen 5 1400 / Intel i5-6600
• RAM: 8 GB
• GPU: AMD RX 470 / NVIDIA GTX 960 / Intel Arc A580 (2 GB VRAM)
• Storage: SSD, 102 GB free
• Internet: Broadband required (even for Campaign)

Recommended

OS: Windows 10/11 64-bit
CPU: AMD Ryzen 5 1600X / Intel i7-6700K
RAM: 12 GB
GPU: AMD RX 6600XT / NVIDIA GTX 1080Ti / RTX 3060 (8 GB VRAM)
Storage: SSD, 102 GB free

PlayStation 5

3. Xbox (Series X|S & One)


Xbox Series X

• Storage: ~100–150 GB
• Performance: Up to 4K 60FPS (Dynamic 120Hz in some modes)
• Features: Dolby Atmos support, Quick Resume, Smart Delivery
• Game Pass Ultimate recommended (for base game + streaming)

Xbox Series S

• Storage: ~100–150 GB
• Performance: Up to 1440p/60FPS (Dynamic scaling)
• Same features as Series X but at lower resolution

Xbox One / One X

• Storage: ~100 GB+
• Performance:
• One S → 1080p 30FPS (dynamic)
• One X → Up to 1440p 30–60FPS (dynamic)
• Xbox Live Gold / Game Pass Ultimate needed for online multiplayer

___________________________________________________________________


4. Mobile Devices (Cloud Gaming)

Black Ops 6 doesn’t have a native mobile app — it runs on Xbox Cloud Gaming through Game Pass Ultimate. Here are the requirements:

Android

• OS: Android 6.0 or later
• RAM: 4 GB+
• Network: At least 10 Mbps Wi-Fi or mobile data (5 GHz preferred)
• Controller: Xbox, PlayStation, or Backbone (touch controls are not supported for BO6 — controller required)

iOS (iPhone & iPad via Safari browser)

• OS: iOS 14.4 or later
• Network: Same as Android (10 Mbps+ stable)
• Controller: Required

Other Cloud Platforms

• PC (browser), Samsung Smart TVs (2020+), Amazon Fire TV, and Meta Quest VR headsets can also run it via Game Pass app.
• Same requirement: 10 Mbps+ stable internet & controller.
